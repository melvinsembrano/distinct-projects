# Change Log

All notable changes to the "distinct-projects" extension will be documented in this file.

## [0.2.0] - 2025-11-20

### Added
- Live color preview while browsing palette and preset colors
- Automatic color restoration on cancel/escape

## [0.1.0] - 2025-11-18

### Added
- Initial release of Distinct Projects extension
- Custom title bar color assignment per workspace/folder
- Status bar color indicator with quick access to color picker
- Three color selection methods:
  - **Color Palette**: 60+ colors organized by category (reds, oranges, yellows, greens, cyans, blues, purples, pinks, browns, grays)
  - **Preset Colors**: 12 quick-access common colors
  - **Custom Color**: Manual hex color input with validation
- Smart text color contrast (automatically picks white or black based on background brightness)
- Workspace-based color persistence (saved in `.vscode/settings.json`)
- Automatic color application on workspace startup
- Support for both active and inactive title bar states
- Command palette integration: "Distinct Projects: Choose Title Bar Color"
- MIT License
- Extension icon for VS Code marketplace
